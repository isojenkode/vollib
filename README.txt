A copy of the files from the (now defunct) vollib.org repository.
The repository was maintained by Stephan Roettger at the University of Tubingen.

The data sets have been converted to Teem nrrd format using the Teem unu command.

For instance, Frog.nhdr was created by: unu make -h -i Frog.raw.gz -c Frog -t uchar -e gzip -s 256 256 44 -sp 0.5 0.5 1 -o Frog.nhdr
